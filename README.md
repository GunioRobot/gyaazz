Gyaazz
======

Outline processing wiki-engine like [Gyazz](http://gyazz.com/).

Setup
-----

Clone gyaazz source-code.

    %git clone git://github.com/shokai/gyaazz.git

Install Dependencies

    # Mac OS X
    %sudo port install tokyocabinet
     
    # Install bundle 
    % bundle install

Run Gyaazz

    % bundle exec ./main.rb
